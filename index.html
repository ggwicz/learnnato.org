The NATO phonetic alphabet is useful and easy to learn.

Instructions

  1. A random letter will appear (for example, "A").
  2. Enter your guess for the code word for this letter (for example, "alpha").
  3. If your guess is correct, another letter will appear.
  4. If you finish the whole alphabet, you win!

Tips

  • There's some flexibility with the spelling of your guesses, as the important thing is how a code word sounds.
  • For example, "alpha" and "alfa" would both be accepted as code words for "A".
