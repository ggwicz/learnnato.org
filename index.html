The NATO phonetic alphabet is useful and easy to learn.

Instructions

  1. A random letter will appear, like "A".
  2. Enter your guess code word for this letter.
  3. If your guess is correct, another letter will appear.
  4. If you finish the whole alphabet, you win!

Tips

  • There is some flexibility with the spelling of your guesses, as the important thing is how a code word sounds.
  • For example, "alpha" and "alfa" would both work as code words for "A".
